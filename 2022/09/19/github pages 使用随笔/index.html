github pages 的基础流程

  1. 安装 hexo。（如果已安装，可以跳过这一步）

    • 先安装 NodeJs （内置 npm 工具）和 git 工具
    • 命令行运行 $ npm install -g hexo-cli 安装 hexo

  2. check out https://github.com/cloutn/cloutn.github.io.git

  3. 切换分支到 master

  4. 打开 source_posts 目录，用 typora 软件编辑 md 文件

具体详细用法可以参考官方的文档：
